# Bytes

Multi-keyword study files grouped by topic. Each keyword is filled to
600-1,500 words across 14 sections, optimized for Knowledge Depth and
Learning Progression.

## How to read

1. Pick a topic from the sidebar.
2. Open the topic's `index.md` for the sub-topic file table.
3. Each keyword inside a file states its `Prereq` and `Next` titles
   in section 12 (Learning Ladder).

See [BYTES_PROMPT v1.0](/bytes/_config/BYTES_PROMPT/) for the master
generation specification.
